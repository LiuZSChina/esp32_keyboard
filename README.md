## **JOKER60**
`Joker 60` 
### **简介**
Joker60是一把poker配列61键键盘，具有一个fn键和一个pn键。
fn键用于输入第二层的键值，如<kbd>F1</kbd>、<kbd>ESC</kbd>（当左上角键被设置为<kbd>`</kbd>时）

### **快捷键**
+ 开启、关闭内置LED灯：<kbd>Pn</kbd> + <kbd>L</kbd>
+ 重置倒计时：<kbd>Pn</kbd> + <kbd>C</kbd>
+ 连接模式转换：<kbd>Pn</kbd> + <kbd>M</kbd> 并且保持三秒以上
